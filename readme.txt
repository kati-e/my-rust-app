Hi there, this is my first attempt at Rust.
Is a little crab holding a weather forecast console app.

To play, type in the terminal: 'cargo run {city} {country_code} (Example: 'cargo run Brisbane AU')
**NOTE: You will need your own API key from OpenWeather -- they are free so go grab one ^^

Tutorials/Documentations/References that helped when first creating this project: 
https://www.rust-lang.org/learn/get-started
https://www.youtube.com/watch?v=DQnLQznJK1Q

Weather data provided by OpenWeather
